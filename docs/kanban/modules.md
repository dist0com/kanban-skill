# Modules

If a line here disagrees with the repo you just read, fix the line.

- **skill** — the kanban skill itself: the board format, the `kanban.mjs` script, and the references the agent follows. `skill/` (installed in this repo via the `.claude/skills/kanban` symlink; packaged as a Claude Code plugin in `.claude-plugin/`).
- **local-ui** — the browser UI to drive the board from buttons, published as `kanban-skill-ui`. `kanban-ui/`.
- **site** — the marketing and landing site, deployed to Cloudflare Pages. `web/`.
- **docs** — the user guides. `docs/guides/`.
